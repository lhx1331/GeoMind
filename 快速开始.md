# 🚀 快速开始 - GeoMind 配置

- **LLM**: DeepSeek (¥10/月)
- **VLM**: 智谱 GLM-4V (¥40-80/月)
- **地图**: OpenStreetMap (免费)
- **总成本**: ¥50-90/月

---

## ✅ 3 步完成配置

### 步骤 1: 准备 API Keys (5 分钟)

#### 1.1 DeepSeek API Key
```
申请地址: https://platform.deepseek.com/
操作: 注册 → API Keys → 创建 → 复制
```

#### 1.2 GLM-4V API Key
```
申请地址: https://open.bigmodel.cn/
操作: 注册 → API管理 → 创建 → 复制
状态: ✅ 您已重置 API Key
```

### 步骤 2: 填写配置文件 (2 分钟)

编辑 `config.yaml` 文件：

```yaml
# 第 11 行
deepseek_api_key: "粘贴您的DeepSeek_Key"

# 第 22 行  
glm_api_key: "粘贴您重置后的GLM_Key"
```

### 步骤 3: 下载 GeoCLIP 模型 (10 分钟)

```bash
# 方式 1: Git LFS (推荐)
git clone https://huggingface.co/geolocal/StreetCLIP ./models/geoclip

# 方式 2: 如果没有 Git LFS
# 访问 https://huggingface.co/geolocal/StreetCLIP
# 手动下载所有文件到 ./models/geoclip/
```

## ❓ 常见问题

**Q: 为什么选择这个组合？**

A: DeepSeek 性价比最高 (约 GPT-4 的 1/50)，GLM-4V 中文理解好且价格合理，免费地图服务适合开发测试。

**Q: 可以换成其他模型吗？**

A: 可以！修改 `config.yaml` 中的 `provider` 即可切换。

**Q: 什么时候需要升级？**

A: 当您需要：
- 更准确的地图数据 → 升级到高德/Google Maps
- 更强的 LLM → 升级到 GPT-4
- 云沙盒 → 升级到 E2B
